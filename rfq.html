IsoDAR (Isotope Decay at Rest Experiment), Prof. Janet Conrad, MIT
2018 - 2020

IsoDAR is designed to perform searches for sterile neutrino oscillations and non-standard neutrino interactions. It does so by accelerating dihydrogen molecules through a radio frequency quadruple (RFQ), then a cyclotron, which then impinge onto a beryllium and lithium target, ultimately producing a large amount of antineutrinos which can be studied. More information about the experiment can be found here, or in this paper.

My role in this experiment was to produce an end-to-end simulation package for the RFQ that could:

  • Be modular with previous experiment simulation packages.
  • Accurately simulate a customizable RFQ factoring in space-charge, which other RFQ simulators could not.
  • Be easily instantiated, customized, and fit in with existing simulations.
  • Be able to be parallelized, and have fast performance.

All of the above were achieved, producing an RFQ package using the WARP particle-in-cell framework that could easily be used and was ~2x faster, and more accurate than preexisting software. I also included built-in diagnostic tools from scratch, which allowed the user to examine beam quality and other features during, and after the simulation. A paper about the RFQ can be read here, and the code can be found here.
